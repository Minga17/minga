Dev Daily Plan

A personal web app that tracks my daily routine — every study session, every skill practiced, every step taken toward my goals. Built with React, JSX, and Tailwind CSS: one of the most popular combinations in modern frontend development today. React powers the interactivity (switching days, checking off tasks), JSX lets me write HTML-like code directly inside JavaScript, and Tailwind handles all the styling. The useState hook tracks which day is selected and which tasks are completed in real time. This app is being built to grow alongside me — mapping daily actions directly to my future career as a developer and IT professional.
